# One-Stop Developer Platform — Vision

## Goal
Create a unified platform that combines:
- Blog (IT consulting, QA, AI, engineering)
- Open-source repositories
- Small apps & utilities
- Gaming section (local multiplayer)
- React Native (Expo) mobile app

## The Platform Acts As:
- Personal brand & Developer portfolio
- Indie product hub & Experiment playground
- Consulting showcase

## Tech Stack
- Next.js / React (Frontend)
- Tailwind CSS (UI)
- Cloudflare Pages (Deployment)
- Gemini CLI / Code Assist (AI Development)
