# Nexus Mobile - iOS
## Tech Stack: Swift, SwiftUI

This is the fully native iOS client for the TechnoNexus ecosystem.

### Architecture
- UI Framework: SwiftUI
- Backend: Supabase Swift SDK
- Real-time: PeerJS (WebRTC) compatible native client
- Styling: Custom brand theme (Neon Cyan, Electric Violet, Dark Mode)

### Getting Started
1. Open `.xcodeproj` in Xcode.
2. Run `pod install` (if using CocoaPods) or resolve Swift Packages.
3. Build and Run.
