# Nexus Mobile - Android
## Tech Stack: Kotlin, Jetpack Compose

This is the fully native Android client for the TechnoNexus ecosystem.

### Architecture
- UI Framework: Jetpack Compose
- Backend: Supabase Kotlin SDK
- Real-time: PeerJS (WebRTC) compatible native client
- Styling: Material 3 + Custom Brand Theme

### Getting Started
1. Open in Android Studio.
2. Sync Gradle.
3. Build and Run.
